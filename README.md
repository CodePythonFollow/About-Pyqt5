### 一、PyQt5包介绍

+ PyQt5.QtWidgets                         各种控件用来布局       如按钮窗口等

+ PyQt5.QtGui                                 关于展示有关类           如画笔、笔刷、颜色、字体、图片等

+ PyQt5.QtCore                               关于各种工具               如正则工具、日期工具等

  + 这里面有个**Qt**                 里面包含各种常量       如颜色常量、对齐方式、排序常量等

+ PyQt5.QtPrintSupport                与打印有关的所有控件、窗口等
